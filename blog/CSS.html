CSS

CSS es el acrónimo para Cascading Style Sheets, que como su nombre lo dice le da estilo visual a las hojas en priorización de cascada. Usualmente se usa este lenguaje para describir la presentación de documentos escritos en HTML o XML.

Dentro del CSS se puede modificar cómo se ve, cómo están dispuestos los elementos, colores, tipografías, márgenes, entre otras cosas.

Términos de CSS

A continuación se describen algunos conceptos clave que te ayudarán a comprender cómo funciona CSS::

  1. Selectores:

    Son instrucciones que dicen a qué elemento HTML se le aplicará un estilo. Si el estilo se repite para varios selectores, se pueden listar con comas. Hay tres tipos:

    • Selector de tipo: Selecciona todos los elementos de un tipo específico.

      Archivo HTML:

      
<p>Este es el primer párrafo.</p>
<p>Otro párrafo.</p>

      Archivo CSS:

      
p {
  font-family: Arial, sans-serif;
  color: blue;
}

    • Selector de clase: Se aplica a los elementos con cierta clase class="...". Se escribe con un punto . al inicio.

      Archivo HTML:

      
<p class="importante">Este párrafo es muy importante.</p>
<p>Este párrafo no tiene clase.</p>

      Archivo CSS:

      
.importante {
  color: red;
  font-weight: bold;
}

    • Selector de ID: Se aplica a un solo elemento con un atributo id="...". Se escribe con # al inicio.

      Archivo HTML:

      
<h1 id="titulo-principal">Este es el título</h1>

      Archivo CSS:

      
#titulo-principal {
  color: darkgreen;
  text-align: center;
  text-transform: uppercase;
}
  2. Propiedades y valores:

    Las propiedades son los estilos que quieres aplicar como el color, tamaño de fuente, altura, ancho, etc. Van seguidas de :.

    Los valores son lo que decides para esa propiedad y deben ir seguidos por ;.

    Ejemplo de archivo CSS:

    
article, aside, details, figcaption, figure, footer, header, hgroup, menu, nav, section {
  display: block;
}

body {
  line-height: 1;
}

ol, ul {
  list-style: none;
}

blockquote, q {
  quotes: none;
}

blockquote:before, blockquote:after,
q:before, q:after {
  content: '';
  content: none;
}

table {
  border-collapse: collapse;
  border-spacing: 0;
}
  3. Conectar HTML con CSS:

    Para conectar un archivo CSS externo al HTML, se utiliza la etiqueta <link> dentro del <head>.

    Ejemplo:

    
<head>
  <link rel="stylesheet" href="main.css">
</head>

Ejemplo overflow:

La propiedad overflow en CSS puede tomar varios valores que determinan cómo se maneja el contenido que se sale del área visible de un contenedor. El valor visible permite que el contenido desbordado se muestre sin restricciones, mientras que hidden oculta todo lo que excede el tamaño del contenedor. Por otro lado, scroll agrega siempre barras de desplazamiento, tanto si el contenido se desborda como si no, y auto muestra barras de desplazamiento solo cuando el contenido realmente sobrepasa los límites del elemento. Estos valores permiten controlar la apariencia y usabilidad de los contenedores con contenido dinámico o extenso.
