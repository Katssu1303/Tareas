CSS

CSS es el acrónimo para Cascading Style Sheets, que como su nombre lo dice le da estilo visual a las hojas en proirización de cascada, usualmente se usa este lenguaje para describir la presentación de documentos escritos en HTML o XML.

Dentro del CSS se puede modificar cómo se ve, cómo están dispuestos los elementos, colores, tipografías, márgenes, entre otras cosas.

Terminos de CSS

A continuación:

    Selectores:

    Son instrucciones que dicen a qué elemento HTML se le aplicará un estilo. Si el estilo se repetira para varios selectores se pueden listar con comas para no tener que hacer uno por uno.

    Propiedades y valores:

    Las propiedades son los estilos que quieres aplicar como el color, tamaño de fuente, height, width y van seguidos de ":".

    Los valores son lo que decides para esa propiedad, es decir como se va a comportar esa propiedad y debe ser seguido por ";".

    Este es un ejemplo de como se vería un archivo CSS 

     
                article, aside, details, figcaption, figure, footer, header, hgroup, menu, nav, section {
                  display: block;
                }

                body {
                  line-height: 1;
                }

                ol, ul {
                  list-style: none;
                }

                blockquote, q {
                  quotes: none;
                }

                blockquote:before, blockquote:after,q:before, q:after {
                  content: '';
                  content: none;
                }

                table {
                  border-collapse: collapse;
                  border-spacing: 0;
                }

    Conectar HTML con CSS:

    Después de tener los selectores y estilos, se usas en el HTML para que se apliquen correctamente.
